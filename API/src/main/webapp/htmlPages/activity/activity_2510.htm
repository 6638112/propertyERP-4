活动时间：

2014.12.19-2015.3.18

活动时间：

【抽最低折扣，享免费物业，喜上加喜再送伊利金典牛奶】12月19日-3月18日期间把抽取的最低折扣以截图形式发到唠一唠，我们将在每月19日评选两个华海最低折扣。最低折扣不仅享受超低物业管理费，还将获得解放区送到家的伊利金典牛奶一箱。（如最低折扣有多人，以先发布截图到街坊的优先）

活动规则：

1、12月19日之前的抽奖折扣不能参加本次活动；

2、如果同一数值最低折扣有多个，以先发布截图到街坊唠一唠的优先；

3、赠送的伊利金典牛奶我们将在活动结束后的五个工作日内（春节期间活动评奖和派发延迟到3月1日）送到获奖者的小区。
