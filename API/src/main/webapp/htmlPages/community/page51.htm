华人在罗马尼亚跳起广场舞 当地人欣然加入

2014-08-01 来自网络

国际在线消息：据新华社国际新闻即时播报“新国际”消息，中国大妈在海外跳广场舞不招待见？也不尽然。比如罗马尼亚首都布加勒斯特的女子健身队不仅给华人带来欢乐，还吸引了当地人一起跳起来。红龙健身队是布加勒斯特目前4支健身队中的一支。4个健身队分别活跃在不同的华人聚集区。其中，东方建材市场健身队成立于五六年前，资历最老。而红龙健身队成立才4个多月，却成员最多，因为大家住得相对集中。

每天晚上6：30分，红龙健身队开始活动。只要不下雨，队员们都会像提前来广场，一边等待一边交谈，分享信息、感受，相互安慰鼓励。

25日傍晚，罗马尼亚首都布加勒斯特东北郊的红龙市场广场的一角十分热闹。数十名身穿黑白条纹上衣和黑色紧身裤的中国女性，三五成群交谈着，并与陆续加入的姐妹亲热地打招呼。

6点30分，队员们伴着欢快的乐曲，十分投入地跳起了广场舞。今天，统一着装显得格外整齐。她们说好，每逢周一、三、五，健身要穿队服。“我们有自己的微信群，现在已有60多人。但因为暑期，不少姐妹陪着孩子外出旅游或回国度假了，”健身队一名召集人爱云告诉记者，来参加健身跳舞的最多时有50多人。这支健身队不光是年轻人，还有好几位奶奶外婆级队员。

说起健身效果，队员们赞不绝口。“锻炼与不锻炼差别很明显，我现在走楼梯不喘了，精神比过去好多了。”“现在生意很难做，大家在一起交流经验，相互鼓励排忧解难。”“过去，晚上打麻将上网看电视，日子过得很无聊，现在早早吃完饭就来广场跟姐妹们一起健身了。”“我们来跳舞很开心，烦恼都忘掉了。”……
