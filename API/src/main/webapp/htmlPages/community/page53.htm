当粤剧遇上小苹果 下沙广场庆中秋

2014-09-09 解放区

受朋友邀请，去围观了下沙广场的中秋晚会，值得一看。老人家的表演和年轻人的表演各有千秋。

中老年人的扇子舞群扇翩翩，腰鼓舞声声鼓响，气势磅礴！

粤剧表演，妆容华丽，动作娴熟，神情生动，妙哉！

年轻人的舞蹈，神曲《小苹果》跳得俏皮有趣，引起台下阵阵欢呼！！

《坏姐姐》的飞吻，引人么么哒！还有那闪亮亮的韩国街舞，怒赞！

后来得知，上台表演的，除了有专业的演员，还有零舞蹈基础的新手。他们因为喜欢舞蹈，于下沙广场结缘！

深入了解广场舞，你会发现，通过广场舞，不管是老人还是年轻人，都可以锻炼身体，拓宽圈子！广场舞还能为各种想学又没有条件的人提供学习舞蹈的机会，甚至是上台表演展现自我的机会！

舞友们，广场上见了！
