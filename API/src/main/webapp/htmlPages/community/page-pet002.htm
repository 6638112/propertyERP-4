猫咪不配合训练该咋办

2014-08-19 解放区

家里如果养有宠物，很多身为主人的人都要训练宠物的，不管宠物是否配合，最终都是要达到主人训练的目的。家里养了猫的朋友就会经常训练他家的猫咪，可是经常遇到猫咪不配合训练该咋办呢?

其实宠物和人的相处能否合平也和主人的训练有很大的关系，所以有的主人在选择宠物的时候会选择比较小一点的宠物，大约一两个月大的最为合适，因为训练起来比较容易的嘛。如果遇到猫咪不配合训练该咋办，相信每个主人都有自己的一套方法。

一般在训练宠物时会用到强迫，引诱加上奖励这几大手段，如果猫咪不配合训练该咋办?主人可以用这三种手段中的前两种，强迫加上引诱，相信没有训练不成的猫咪。当然如果在训练中表现的非常好的猫咪，主人也要给予一定的奖励才行的哦。
